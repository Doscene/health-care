# 10 发现与知识社区需求

## 1. 功能概述

为年轻患者提供健康知识库、并发症风险评估、同龄人社群入口。**仅年轻患者角色可见此 Tab**，其他角色无此页面。

---

## 2. 用户故事

| 角色 | 场景 | 期望结果 |
|------|------|----------|
| 年轻患者 | 备孕期间想知道降压药能不能吃 | 发现页进入备孕专区，看到用药安全知识和孕期注意事项 |
| 年轻患者 | 想知道自己的10年心血管风险有多大 | 并发症风险页面展示动态趋势，看到自己风险在下降 |
| 年轻患者 | 想跟同样高血压备孕的人交流 | 进入"30岁+高血压备孕群"，找到同龄人的经验分享 |
| 年轻患者 | 确诊后焦虑害怕，不敢告诉同事 | 匿名树洞倾诉，获得同龄病友的理解和支持 |

---

## 3. 人生阶段知识库

### 3.1 备孕/孕期专区
- 专区标识：🍼 粉色卡片 + "进行中"角标
- 描述：ACEI类降压药需替换为孕期安全药物，血糖目标自动收紧
- 子标签入口：用药安全 / 血糖控制 / 产检数据
- 点击 → 进入对应知识列表

### 3.2 职场应对指南
- 专区标识：💼 灰色卡片
- 描述：应酬场景如何低调控糖？加班与血压波动的应对策略
- 子标签入口：应酬技巧 / 职场饮食

### 3.3 运动专区
- 专区标识：🏃
- 描述：高血压+糖尿病患者的运动处方，安全有效
- 子标签入口：有氧运动 / 力量训练

### 3.4 内容形式
- 图文文章（内置 CMS 内容）
- 5 分钟短视频（后续迭代）
- 分类筛选 + 搜索

---

## 4. 并发症风险评估

### 4.1 核心指标
- **10 年心血管疾病风险**
  - 基于 Framingham/China-PAR 模型计算
  - 展示当前值 + 历史趋势
  - 示例：8.2% → 7.5%（下降 0.7%）

### 4.2 影响因素展示
- 可改善因素：血压、血糖、体重、吸烟
- 不可改变因素：年龄、性别、家族史
- 用雷达图或进度条展示"改善空间"

### 4.3 风险变化追踪
- 每次更新指标后重新计算
- 季度趋势曲线
- 鼓励性文字："比上季度降低 0.7%，继续保持！"

## 5. 同龄人社群

### 5.1 社群入口
- 发现页底部"同龄人社群"模块
- 群组列表按病种+年龄段分组

### 5.2 群组类型（MVP 预置）

| 群组名称 | 目标人群 | 规模示例 |
|----------|---------|---------|
| 30岁+高血压备孕群 | 备孕中年轻高血压患者 | 128人 |
| 程序员糖友自救会 | IT行业年轻糖尿病患者 | 256人 |
| 新手妈妈控糖互助组 | 产后糖尿病患者 | 待上线 |
| 北上广高血压职场群 | 一线城市高压职场人群 | 待上线 |

### 5.3 社群功能（MVP 基础版）
- 群组简介 + 成员数 + 今日消息数
- 文字发帖 + 评论（基础论坛形式）
- "热聊"标签（活跃度标记）
- 匿名发帖/回复（可选开关）

### 5.4 匿名树洞
- 独立入口："说点不敢说的"
- 完全匿名发帖
- 同龄人回复支持
- 内容审核（敏感词过滤）

### 5.5 家庭单位入组（后续迭代）
- 患者和主要照护者一起加入
- 分享照护经验

## 7. 页面/交互说明

### 7.1 发现页整体布局
- 顶部标题"发现" + 副标题"知识 · 风险 · 社群"
- **人生阶段知识库**：纵向卡片列表，每个卡片含图标+标题+描述+子标签
- **并发症风险评估**：数据卡片 + 环形图
- **同龄人社群**：群组卡片列表

### 7.2 知识详情页
- 标题 + 发布时间
- 图文内容
- 收藏按钮
- 相关推荐

### 7.3 社群群组页
- 群组头图 + 简介
- 帖子列表（标题+作者+回复数+时间）
- 发帖按钮 / 匿名发帖开关
- 热门/最新 Tab 切换

## 8. 数据结构要点

```typescript
KnowledgeArticle {
  id: string
  title: string
  category: 'pregnancy' | 'career' | 'exercise'
  subCategory: string
  content: string
  coverImage?: string
  tags: string[]
  publishedAt: datetime
  viewCount: number
}

ComplicationRisk {
  userId: string
  calculatedAt: datetime
  cardiovascularRisk10y: number
  previousRisk: number
  change: number
  factors: {
    modifiable: { bpEffect: number, bgEffect: number, bmiEffect: number }
    nonModifiable: { ageEffect: number, familyEffect: number }
  }
}

CommunityGroup {
  id: string
  name: string
  description: string
  memberCount: number
  todayMessageCount: number
  tags: string[]
  isActive: boolean
}

CommunityPost {
  id: string
  groupId: string
  authorId?: string          // 匿名时为 null
  isAnonymous: boolean
  title: string
  content: string
  replyCount: number
  createdAt: datetime
}

CommunityReply {
  id: string
  postId: string
  authorId?: string
  isAnonymous: boolean
  content: string
  createdAt: datetime
}
```

## 9. 接口需求概要

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 知识文章列表 | GET | /api/knowledge | query: category, page |
| 知识文章详情 | GET | /api/knowledge/:id | |
| 并发症风险评估 | GET | /api/complication-risk | query: userId |
| 社群群组列表 | GET | /api/community/groups | |
| 社群帖子列表 | GET | /api/community/groups/:id/posts | query: sort, page |
| 发帖 | POST | /api/community/groups/:id/posts | body: { title, content, isAnonymous } |
| 帖子详情 | GET | /api/community/posts/:id | |
| 回复帖子 | POST | /api/community/posts/:id/replies | body: { content, isAnonymous } |

## 10. 异常处理

| 场景 | 处理 |
|------|------|
| 知识文章加载失败 | 展示加载失败占位，点击重试 |
| 风险计算数据不足 | 提示"需要至少 X 周数据才能评估，请持续记录" |
| 社群帖子被举报/违规 | 后台审核，违规内容屏蔽 |
| 匿名发帖违规 | 仍可追溯（后端记录真实userId），严重违规封禁 |
