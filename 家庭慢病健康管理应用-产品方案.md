# 家庭慢病健康管理应用 - 产品方案

## 一、产品定位

**一句话定位**：高血压和糖尿病家庭的日常健康搭子——让记录不费劲，让关心不打扰。

**核心价值**：
- 聚焦高血压 + 糖尿病两个病种
- 服务三类家庭：子女远程关怀父母、退休夫妻互助照护、年轻已婚患者与配偶共同管理
- 现阶段不考虑与硬件合作，纯软件方案

---

## 二、核心用户画像

### 画像 A：远程子女（主动关怀者）

| 维度 | 描述 |
|------|------|
| **人物** | 32 岁，一线城市工作，父母在家乡/同城不同住 |
| **痛点** | 每次打电话问"最近血压怎么样"，父母都说"挺好的"，实际漏药、少测、饮食失控 |
| **诉求** | 用**最低打扰**的方式知道父母真实状况；在父母不愿说/说不清时能及时介入 |

### 画像 B：退休夫妻（互助照护者）

| 维度 | 描述 |
|------|------|
| **人物** | 65 岁左右，一方高血压、另一方糖尿病，或一人同时患两病 |
| **痛点** | 药名记不住、测了血压写在烟盒上丢了、去医院像打仗、不知道两个病怎么兼顾着吃 |
| **诉求** | **少打字、少思考**，最好是"拍个照、说句话"就能搞定，两个人能互相提醒 |

### 画像 C：年轻的已婚患者（共同管理者）

| 维度 | 描述 |
|------|------|
| **人物** | 28-40 岁，已婚，刚确诊或确诊数年，职场中坚，配偶为首要支持人 |
| **痛点** | 工作应酬多、饮食失控、作息不规律；害怕并发症和遗传风险；备孕/孕期用药困惑；不想被当成"病人"影响社交形象 |
| **诉求** | **低调高效**地管理疾病，不耽误工作和生活；需要专业指导（备孕、运动、职场社交饮食）；与配偶**协作而非被监管**；长期数据可视化增强掌控感 |

## 三、MVP 核心功能清单

### 3.1 必备闭环：监测 → 记录 → 提醒 → 反馈 → 共享

| 模块 | 功能点 | 设计原则 |
|------|--------|----------|
| **家庭圈** | 创建家庭，邀请成员（子女发起 or 夫妻互邀） | 老人端字体大、步骤少，支持子女代建代管 |
| **智能用药** | ① 拍药盒/说明书 AI 识别药品<br>② 自动生成服药计划（早/中/晚/睡前）<br>③ 到点提醒 + 一键确认（语音/点击）<br>④ **多药冲突检测**（如 ACEI 类降压药与二甲双胍的肾功监测提醒） | 老人**不录入药名**，只拍盒子 |
| **指标记录** | ① 血压（收缩/舒张/心率）<br>② 血糖（空腹/餐前/餐后 2h）<br>③ 语音快速记录："空腹血糖 7.2"<br>④ 拍照识别血压计屏幕（可选） | 一次记录不超过 10 秒 |
| **家庭看板** | ① **老人端**：今天我还剩几件事（服药/测血压/测血糖）<br>② **子女端**：本周父母的"健康简报"（非实时流水）<br>③ **夫妻端**：两人本周指标对比趋势图<br>④ **年轻患者端**：个人健康趋势报告、并发症风险评估、备孕/运动/职场饮食专区 | 年轻患者需要**专业深度**和**隐私掌控**，数据可匿名分享 |
| **人生阶段适配** | ① 备孕模式：标记备孕/孕期，自动标注孕期安全用药提醒、血糖控制更严格目标<br>② 职场模式：应酬场景快速记录（拍照酒水菜品估算）、加班与血压波动关联分析<br>③ 遗传风险评估：基于家族史生成子女未来风险提示和预防建议 | 不是附加功能，是年轻患者的**核心刚需** |
| **异常响应** | ① 血压 ≥180/110 或空腹血糖 ≥13.9 → 立即提醒患者 + 紧急联系人<br>② 连续 3 天偏离目标区 → 温和建议"最近波动有点大，要不要记录一下饮食/睡眠？"<br>③ 漏服 2 次以上 → 询问原因并调整提醒策略 | 分级响应，避免骚扰 |
| **复诊助手** | ① 复诊倒计时（如"距下次心内科复诊还有 5 天"）<br>② 近期数据自动汇总（近 2 周血压/血糖均值、极值、漏服次数）<br>③ **问医生草稿箱**：AI 基于异常数据生成建议提问的问题 | 导出为一张 A4 纸可直接带去医院 |

### 3.2 差异化亮点（决定产品辨识度）

#### 1. 双病兼容食谱推荐
- 输入"高血压 + 糖尿病"，推荐**既低钠又低 GI** 的家常菜
- 支持"两人份"：一道菜同时满足控糖需求和低盐需求，不用分锅做
- 食材替换建议：想做红烧肉？提示"用代糖替代冰糖，加魔芋块吸油，总量控制在 200g 以内"

#### 2. 夫妻协作任务（适配年轻患者）
- **协作模式切换**：退休夫妻是"提醒-确认"模式；年轻夫妻是"共同目标"模式（如"本月一起把平均收缩压降到 130 以下"）
- 妻子提醒丈夫测血压可获得"默契值"（轻游戏化）
- 设置"互为紧急联系人"，一方异常数据另一方优先收到
- 家庭微运动挑战："本周全家合计走满 50 公里解锁家庭勋章"
- **隐私边界**：年轻患者可设置"工作日数据仅自己可见，周末共享给配偶"，避免工作应酬数据引发日常唠叨

#### 3. 子女的"轻触达"
- 不在父母手机上装"监控软件"，而是父母主动记录 → 子女被动接收"周报"
- 异常时子女收到的是"建议关心"而非"警报"
- 自动周报/月报：生成"爸妈上周健康简报"，以温和图文形式推送

## 四、核心用户旅程设计

### 4.1 旅程 A：远程子女关怀线（以"发现异常"为核心）

#### Day 1-3（Onboarding）
```
├─ 子女安装 App，创建"张家健康圈"，生成邀请码
├─ 帮父母下载，扫码加入，代设置用药计划（拍药盒）
└─ 告诉父母："每天测完血压点一下，我每周收到周报，不用天天汇报"
```

#### 日常（Low Touch）
```
├─ 父母每日：测血压/血糖 → 语音说数字或手动点"已服/已测"
├─ 父母偶尔：拍一张今天的饭菜，App 自动估算盐分和碳水（可选）
└─ 子女每周一：收到"爸妈上周健康简报"（血压平稳/漏药 1 次/下周二复诊）
```

#### 异常（High Touch）
```
├─ 父亲连续 3 天晨起血压 >150/95
├─ 系统：先问父亲"最近睡眠怎么样？盐是不是吃多了？"
├─ 父亲未回应 or 确认不适 → 推送子女："爸爸血压近 3 天偏高，建议电话关心"
├─ 子女电话沟通 → 发现父亲最近擅自减药
└─ 子女通过 App "用药锁定"功能（提醒父亲遵医嘱，或设置更频繁的提醒）
```

#### 复诊
```
├─ App 提前 3 天提醒复诊
├─ 子女远程视频接入（如果支持）或事前把"近期数据报告"微信发给父亲打印带去
└─ 医生看到连续数据，调整方案，子女同步知晓
```

### 4.2 旅程 B：夫妻互助照护线（以"共同管理"为核心）

#### 早晨 7:00
```
├─ 丈夫手机/音箱提醒："该吃降压药了"
├─ 丈夫点击"已服用" or 语音"吃了"
└─ 妻子手机显示：老公已服药 ✓
```

#### 上午 10:00
```
├─ 妻子测空腹血糖：语音"空腹血糖 6.8"
└─ App 记录并提示："很好，在目标范围内"
```

#### 中午 12:00
```
├─ 打开 App "今天吃什么"
├─ 输入：高血压 + 糖尿病，2 人份
└─ 推荐：清蒸鲈鱼 + 杂粮饭 + 凉拌黄瓜。提示："鲈鱼不要浇蒸鱼豉油，用葱姜蒜代替"
```

#### 下午 6:00
```
├─ 丈夫提醒妻子："该测餐后血糖了"
├─ 妻子记录：餐后 2h 血糖 9.0
└─ 系统：正常，但比昨天高，建议记录今晚主食是否偏多？
```

#### 睡前
```
├─ 查看"今日家庭健康日历"：两人 4 项任务全部完成
└─ 解锁"默契家庭"小徽章（轻激励）
```

#### 每月底
```
├─ 生成《家庭健康月报》
├─ 包含：两人血压/血糖趋势图、服药依从性、复诊记录
└─ 下次复诊直接出示给医生，或子女远程查看
```

### 4.3 旅程 C：年轻已婚患者共同管理线（以"人生阶段适配"为核心）

#### 确诊初期（Week 1-2）
```
├─ 患者确诊高血压/糖尿病，焦虑感强
├─ 邀请配偶加入家庭圈，但强调"是共同管理不是被监督"
├─ 开启"新手指引"：解释疾病、破除误解（"得了糖尿病不等于什么都不能吃"）
├─ 设置个人目标：如"3 个月内糖化血红蛋白降到 7% 以下"
└─ 配偶端接收"支持指南"：如何提醒不招人烦、如何在聚餐时帮患者打掩护
```

#### 职场日常（工作日）
```
├─ 早晨 8:00：手机提醒服药（可设置极低调提醒，如仅震动+简短文字）
├─ 中午 12:00：外卖/食堂拍照记录，AI 估算碳水和盐分，给出"这顿安全/注意/超标"评级
├─ 晚上 8:00（应酬场景）：快速记录喝了什么酒、吃了什么菜，App 提示"今晚建议加测一次血压"
├─ 睡前：查看今日数据，系统关联分析"本周三次应酬，平均次日晨起血压高 15mmHg"
└─ 配偶端：仅看到"今日已完成记录"，不看到具体饮食细节（保护隐私）
```

#### 备孕/孕期阶段（关键差异化场景）
```
├─ 开启"备孕模式"：用药方案自动标注孕期安全性（如 ACEI 类降压药需替换）
├─ 血糖目标自动收紧（空腹 <5.3，餐后 <6.7）
├─ 推送"准爸爸/准妈妈慢病管理"专题知识
├─ 配偶端强化同步：此阶段数据对配偶完全透明，共同承担管理责任
├─ 产检数据手动录入或拍照识别，与日常血糖/血压数据形成完整孕期档案
└─ 产后自动切换至"哺乳期模式"，更新用药安全提醒
```

#### 长期管理（季度复盘）
```
├─ 生成《季度健康趋势报告》：不是简单罗列数字，而是"故事化"呈现
│   └─ 例："这个夏天你控制了 12 次应酬中的 8 次，平均血压比春季下降 8mmHg"
├─ 并发症风险评估更新：基于最新数据估算 10 年心血管风险变化
├─ 遗传风险页：为未来子女提供预防建议（如"建议孩子成年后每年测一次空腹血糖"）
├─ 夫妻共同设定下季度目标，解锁"健康合伙人"进阶徽章
└─ 
```

## 六、情感与尊严保护机制

慢病老人最怕的是**"被当成病人，失去自主权"**。

### 6.1 患者自主权设计
- **可见性分级**：患者可以设置"哪些数据对子女可见"。例如：血压对女儿可见，但体重和饮食记录仅自己可见
- **年轻患者隐私保护**：可设置"工作日应酬记录仅自己可见""血糖波动细节不实时同步配偶"，避免过度监控感；异常时才触发共享
- **成就正向激励**：不是提醒"您又忘了吃药"，而是庆祝"您连续 30 天按时服药，血压控制很棒，解锁'自律之星'"
- **人生故事模块**：鼓励记录年轻时的故事、家庭照片，**慢病只是人生的一小部分，不是全部**
- **年轻患者专属**："掌控感"叙事——强调"你在主动管理自己的人生，而不是被疾病定义"

### 6.2 照护者支持（常被忽视）
- **照护者疲惫指数测评**：定期评估主要照护者（通常是配偶或女儿）的身心状态
- **喘息服务资源**：当照护者疲惫指数过高时，推荐本地上门护理、短期托老机构等资源
- **照护技巧微课**："如何帮助卧床老人翻身""糖尿病足的日常检查"，5 分钟短视频

## 七、风险兜底：从管理到急救

### 7.1 渐进式风险响应

| 级别 | 触发条件 | 响应动作 |
|------|----------|----------|
| **绿色** | 数据正常 | 自动记录 |
| **黄色** | 单项指标异常（如单次血糖偏高） | 推送科普 + 建议复测 |
| **橙色** | 多日异常或症状组合 | 建议联系家庭医生/复诊，并通知紧急联系人 |
| **红色** | 疑似心梗、低血糖昏迷等 | 触发一键呼救 + 自动发送定位和健康档案给 120 |

### 7.2 家庭急救包管理
- 记录家庭常备药和急救用品有效期
- 急救情景指南："如果爸爸突然胸痛，先做什么？打 120 时说什么？硝酸甘油怎么舌下含服？"

## 八、冷启动与留存抓手

| 阶段 | 策略 |
|------|------|
| **第一周** | 子女帮父母设置好，完成首次记录即送"家庭健康档案"电子封面 |
| **第一周（年轻）** | 完成疾病认知测试 + 设置人生阶段目标，解锁"健康合伙人"身份 |
| **第一个月** | 连续记录 7 天解锁"健康习惯勋章"；夫妻组队完成周目标解锁双人皮肤 |
| **第一个月（年轻）** | 首次生成《季度趋势故事报告》并支持分享（朋友圈匿名海报），强化成就感 |
| **持续留存** | 每周一早上 8 点推送"上周家庭健康简报"，成为固定触点 |
| **持续留存（年轻）** | 每月推送"并发症风险变化"和"人生阶段提醒"（如备孕模式到期提醒） |
| **异常介入** | 用"关心"而非"警告"的语气推送，降低老人抵触感 |
| **异常介入（年轻）** | 用"数据洞察"语气替代（如"这周三次应酬后次日血压都高了，试试减少啤酒？"） |

## 九、社会支持网络（后续迭代）

### 9.1 病友家庭小组
- **同病种+同地域+同年龄段**匹配：如"北京朝阳区 60-70 岁二型糖友家庭"小组
- **年轻患者专属社群**："30 岁+高血压备孕群""程序员糖友自救会""新手妈妈控糖互助组"
- **家庭单位入组**：患者和主要照护者一起加入，分享"怎么劝老爸测血糖不吵架"等实操经验
- **匿名倾诉空间**："今天给妈妈测血糖她又摔了血糖仪，好累"——这类情绪在家人面前难以表达，但在病友家属圈能获得理解
- **年轻患者匿名树洞**："确诊两年不敢告诉同事""怕遗传不敢生孩子"——同龄人的共鸣和成功案例最具说服力

### 9.2 医生-患者-家庭三角沟通
- **医生端轻量级入驻**：不是在线问诊，而是医生设置"本月重点"（如"夏季出汗多，注意低血糖"），推送给绑定他的患者家庭
- **家庭问诊陪同**：子女可远程加入父母的视频复诊（三方通话），帮记录医嘱、帮提问
- **医嘱结构化**：医生说的"少吃多餐、少油少盐"自动转化为 App 内的可执行任务和饮食标签

### 9.3 社区健康资源地图
- **本地资源聚合**：附近哪个社区医院可以测糖化血红蛋白？哪里的健走队适合老年人？哪个药店有这个慢病品种的集采药？
- **家庭健康活动**：社区义诊报名、医院健康讲座提醒、病友家庭线下聚餐（非强制）
