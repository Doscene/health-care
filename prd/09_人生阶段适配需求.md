# 09 人生阶段适配需求

## 1. 功能概述

针对年轻慢病患者在不同人生阶段（备孕/孕期、职场、遗传风险）的特殊需求，提供模式切换、目标调整、知识推送等功能。**年轻患者的差异化刚需，非附加功能。**

---

## 2. 用户故事

| 角色 | 场景 | 期望结果 |
|------|------|----------|
| 年轻患者（备孕） | 确诊高血压后备孕，担心药物影响胎儿 | 开启备孕模式后自动标注孕期安全用药，血糖目标收紧 |
| 年轻患者（职场） | 工作应酬多，想知道喝酒对血压的影响 | 应酬场景快速记录，系统关联分析"应酬-血压"数据 |
| 年轻患者（遗传） | 担心孩子以后也会得糖尿病 | 基于家族史生成遗传风险评估和预防建议 |
| 配偶 | 备孕期间想全力协助 | 备孕模式下数据完全透明，共同承担管理责任 |

---

## 3. 备孕/孕期/哺乳期模式

### 3.1 模式流转
```
备孕模式 → 确认怀孕 → 孕期模式 → 确认分娩 → 哺乳期模式 → 回归日常
```

### 3.2 备孕模式
| 维度 | 变化 |
|------|------|
| 用药安全 | 自动标注孕期药物安全等级（A/B/C/D/X），ACEI类降压药标记为需替换 |
| 血糖目标 | 空腹收紧至 < 5.3 mmol/L，餐后 < 6.7 mmol/L |
| 血压目标 | 收缩压 < 130，舒张压 < 80 |
| 知识推送 | "准爸爸/准妈妈慢病管理"专题知识 |
| 产检数据 | 支持手动录入产检指标（体重、糖耐量等） |
| 数据共享 | 此阶段数据对配偶完全透明 |

### 3.3 孕期模式
- 继承备孕模式全部规则
- 新增孕期周数跟踪
- 增加"孕期体重增长曲线"对照
- 药物调整提醒（不同孕期用药可能不同）

### 3.4 哺乳期模式
- 自动切换 → 更新用药安全提醒（哺乳期禁忌药）
- 血糖/血压目标逐步回归日常

### 3.5 模式切换入口
- 首页顶部"备孕模式"标签 → 点击进入
- "我的" → 人生阶段 → 选择当前阶段

## 4. 职场模式

### 4.1 应酬场景记录
- 首页快捷入口"应酬"（🍷 图标）
- 快速记录表单：
  - 喝了什么酒：啤酒/白酒/红酒/无（快速选择）
  - 吃了什么：拍照记录
  - 应酬时长：约 X 小时
- 保存后提示"今晚建议加测一次血压"

### 4.2 应酬-血压关联分析
- 系统根据记录时间和次日血压数据，自动关联分析
- 洞察卡片："本周三次应酬后次日晨起血压平均高 15mmHg"
- 累计数据：按应酬类型分组展示平均血压影响

### 4.3 低调提醒设置
- 通知方式：仅震动+简短文字
- 不弹出大通知，避免社交场合尴尬
- 可设置"工作时间免打扰"

### 4.4 加班与血压波动
- 可选录入加班时间段
- 系统关联加班与次日血压数据
- 长期形成"加班-血压"关联趋势报告

---

## 5. 遗传风险评估

### 5.1 家族史录入
- 录入父母、祖父母的疾病史
- 支持多病种选择：高血压/糖尿病/冠心病/脑卒中

### 5.2 风险计算
- 基于家族史 + 患者自身数据
- 估算子女未来患病风险等级（低/中/高）
- 展示风险变化趋势（如从 35% 降至 30%）

### 5.3 预防建议
- 针对不同风险等级推送：
  - 饮食建议
  - 运动处方
  - 最佳筛查年龄和频率
- "建议孩子成年后每年测一次空腹血糖"

---

## 6. 运动模式（MVP可含基础版）

### 6.1 运动记录
- 录入运动类型（散步/跑步/游泳/太极等）
- 时长 + 强度（轻度/中度/剧烈）

### 6.2 运动安全提示
- 高血压患者避免剧烈运动提醒
- 糖尿病患者运动前血糖检测提醒
- 运动后血糖/血压变化关联

---

## 7. 页面/交互说明

### 7.1 人生阶段专区（"发现"页内）
- 备孕/孕期专区卡片（粉色）
- 职场应对指南卡片（灰色）
- 运动专区卡片（绿色）
- 每张卡片：图标 + 标题 + 描述 + 子标签

### 7.2 模式切换页（"我的" → 人生阶段）
- 当前阶段高亮
- 备孕模式 → 孕期模式 → 哺乳期模式 → 日常模式
- 切换确认弹窗（提示模式变化影响）

### 7.3 应酬记录
- 快速选择式表单
- 拍照入口
- 保存后关联提醒

## 8. 数据结构要点

```typescript
LifeStage {
  userId: string
  currentMode: 'normal' | 'preconception' | 'pregnancy' | 'lactation'
  pregnancyWeek?: number
  startedAt: datetime
  targetSystolic: number
  targetDiastolic: number
  targetFastingGlucose: number
  targetPostprandialGlucose: number
  medicationSafetyNotes: string[]  // 用药安全提醒
}

SocialRecord {
  id: string
  userId: string
  alcoholType: 'beer' | 'wine' | 'spirit' | 'none'
  mealPhoto?: string
  duration: number            // 小时
  notes?: string
  recordedAt: datetime
  nextDayBp?: { systolic: number, diastolic: number }
}

OvertimeRecord {
  id: string
  userId: string
  date: date
  hours: number
  nextDayBp?: { systolic: number, diastolic: number }
}

GeneticRisk {
  userId: string
  familyHistory: {
    relation: string         // 父亲/母亲/祖父等
    disease: string
    onsetAge?: number
  }[]
  offspringRisk: {
    hypertension: number     // 百分比
    diabetes: number
    overall: 'low' | 'medium' | 'high'
  }
  preventionAdvice: string[]
  updatedAt: datetime
}
```

## 10. 异常处理

| 场景 | 处理 |
|------|------|
| 切换至备孕模式但未录入家族史 | 引导完成家族史录入 |
| 孕期模式中漏记产检 | 根据孕周提醒产检时间 |
| 应酬记录后未测血压 | 次日早晨推送提醒加测血压 |
| 遗传风险数据不足 | 展示"请完善家族史信息以获得更准确评估" |

---

## 11. 验收标准

- [ ] 备孕模式正确开启并调整目标值
- [ ] 孕期用药安全标注正确（ACEI类标记需替换）
- [ ] 备孕→孕期→哺乳期模式流转正常
- [ ] 应酬记录快速表单完整
- [ ] 应酬-血压关联分析数据正确
- [ ] 低调提醒设置生效（仅震动）
- [ ] 家族史录入完整
- [ ] 遗传风险评估计算并展示
- [ ] 预防建议按风险等级差异化推送
- [ ] 产检数据录入和查看正常
- [ ] 首页人生阶段标签点击跳转专区
