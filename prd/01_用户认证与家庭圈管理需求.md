# 01 用户认证与家庭圈管理需求

## 1. 功能概述

用户注册/登录、创建家庭健康圈、邀请家庭成员、角色分配与成员管理。家庭圈是整个应用的数据共享基础单元，支持三种角色：老人（患者）、年轻患者、子女（照护者）。

---

## 2. 用户故事

| 角色 | 场景 | 期望结果 |
|------|------|----------|
| 子女（张小红） | 想远程关心父母健康 | 注册后创建家庭，邀请父母加入，查看父母健康数据 |
| 老人（张建国） | 不太会用智能手机 | 子女代建家庭圈，自己扫码即加入，无需复杂操作 |
| 年轻患者（李想） | 刚确诊高血压，想和配偶共同管理 | 注册后邀请配偶加入，设置协作模式和隐私边界 |
| 配偶 | 收到邀请 | 一键加入家庭圈，开始协作 |

---

## 3. 注册/登录

### 3.1 注册方式
- 手机号 + 验证码注册（主流程）
- 微信授权登录（备选流程）
- 注册时需选择角色：**我是患者** / **我是家人（照护者）**

### 3.2 分角色注册流程

#### A. 子女注册
```
App启动 → 手机号登录 → 选择"我是家人" → 填写姓名 → 进入首页
         → 引导创建家庭 → 邀请父母
```

#### B. 年轻患者注册
```
App启动 → 手机号登录 → 选择"我是患者" → 填写姓名+出生年份
         → 选择病种（高血压/糖尿病/两者都有）→ 进入首页
         → 引导邀请配偶加入
```

#### C. 老人注册
- 老人通常由子女代为操作，老人端只需：
```
App启动 → 点击"加入家庭" → 输入邀请码/扫码 → 确认加入 → 进入老人首页
```
- 无邀请码时的兜底流程：
```
App启动 → 手机号登录 → 选择"我是患者" → 填写姓名+出生年份 → 创建空家庭圈 → 进入首页
```

### 3.3 登录
- 手机号 + 验证码登录（默认）
- 微信授权登录
- 登录态保持：**30天免登录**，过期需重新验证

---

## 4. 家庭圈管理

### 4.1 创建家庭圈
- **发起人**：子女 或 年轻患者
- **创建流程**：
  1. 点击"创建家庭健康圈"
  2. 输入家庭名称（placeholder: "张家的健康圈"）
  3. 自动生成 6 位邀请码（数字+字母组合，如 `A3F8K2`）
  4. 支持分享：微信 / 复制邀请码 / 短信 / 面对面扫码
  5. 创建成功跳转首页

### 4.2 邀请成员
- **邀请方式**：
  - 生成邀请二维码（微信扫码直达）
  - 复制邀请码文本分享到微信
  - 短信直发（含邀请链接）
- **邀请状态流转**：已邀请 → 已加入 / 已过期
- **限制**：
  - 一个家庭圈最多 **10 人**
  - 邀请码有效期 **48 小时**，过期需重新生成

### 4.3 加入家庭圈
- **有邀请码时**：
  1. 在登录页或首页点击"加入家庭"
  2. 输入 6 位邀请码或扫描二维码
  3. 系统展示家庭名称和已有成员（脱敏显示）
  4. 选择本人在家庭中的角色：患者 / 配偶 / 子女 / 其他亲属
  5. 确认加入 → 跳转首页

- **无邀请码时**（兜底）：
  - 老人登录后自动拥有仅自己的空家庭圈
  - 后续可从家庭页主动邀请子女加入

### 4.4 角色与权限矩阵

| 角色 | 查看权限 | 操作权限 | 特殊能力 |
|------|---------|---------|---------|
| 患者本人 | 自己全部数据 | 全部功能 | 设置数据对谁可见 |
| 配偶 | 患者授权数据 | 提醒、查看报表 | 互为紧急联系人 |
| 子女 | 父母授权数据 | 查看周报、代管用药 | 代建代管父母账号 |
| 其他亲属 | 患者授权数据 | 仅查看 | 不可编辑 |

### 4.5 成员管理
- 成员列表：头像 + 姓名 + 角色标签 + 健康状态概要
- 修改备注名（如将"张建国"显示为"爸爸"）
- 修改成员角色
- 移除成员（仅创建者可操作，需二次确认）
- 退出家庭圈（需二次确认，退出后数据不可见）

## 5. 页面/交互说明

### 5.1 注册/登录页
- 手机号输入框 + "获取验证码"按钮（60s 倒计时）
- 微信授权登录入口（底部）
- 角色选择：大卡片（"我是患者"和"我是家人"），图文引导
- 《用户协议》和《隐私政策》勾选

### 5.2 创建家庭圈页
- 家庭名称输入框
- 邀请码展示卡片（大字号、可复制）
- 三个分享按钮：微信 | 复制 | 短信
- 已邀请成员列表（头像 + 状态）

### 5.3 加入家庭圈页
- 6 位邀请码输入框（自动大写）
- "扫一扫加入"按钮
- 家庭预览卡片（名称 + 已有成员头像）
- 角色选择器（患者/配偶/子女/其他）
- "确认加入"按钮

### 5.4 家庭圈成员列表
- 位于「家庭」Tab 内
- 顶部：家庭名称 + 成员总数
- 成员卡片列表
- 底部固定："+ 邀请家庭成员"按钮

## 6. 数据结构要点

```typescript
User {
  id: string
  phone: string
  name: string
  avatar: string
  selfRole: 'elderly' | 'young_patient' | 'child'
  age: number
  diseases: ('hypertension' | 'diabetes')[]
  createdAt: datetime
}

Family {
  id: string
  name: string
  creatorId: string
  inviteCode: string          // 6位，48h有效
  inviteCodeExpireAt: datetime
  memberCount: number
  createdAt: datetime
}

FamilyMember {
  id: string
  familyId: string
  userId: string
  role: 'patient' | 'spouse' | 'child' | 'other'
  nickname: string            // 在家庭中的称呼，如"爸爸"
  joinedAt: datetime
  visibilitySettings: VisibilitySettings
}

VisibilitySettings {
  bloodPressure: 'all' | 'spouse_only' | 'child_only' | 'self_only'
  bloodSugar:    'all' | 'spouse_only' | 'child_only' | 'self_only'
  weight:        'all' | 'spouse_only' | 'child_only' | 'self_only'
  dietRecord:    'all' | 'spouse_only' | 'child_only' | 'self_only'
  socialRecord:  'all' | 'spouse_only' | 'self_only'   // 应酬记录
}
```

## 7. 接口需求概要

| 接口 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 发送验证码 | POST | /api/auth/send-code | body: { phone } |
| 登录/注册 | POST | /api/auth/login | body: { phone, code } 首次自动注册 |
| 微信登录 | POST | /api/auth/wechat-login | body: { code } |
| 创建家庭 | POST | /api/families | body: { name } |
| 重新生成邀请码 | POST | /api/families/:id/invite-code | |
| 加入家庭 | POST | /api/families/join | body: { inviteCode, role } |
| 获取家庭详情 | GET | /api/families/:id | 含成员列表 |
| 更新成员信息 | PUT | /api/families/:id/members/:memberId | body: { nickname, role } |
| 移除成员 | DELETE | /api/families/:id/members/:memberId | |
| 退出家庭 | POST | /api/families/:id/leave | |
| 更新隐私设置 | PUT | /api/users/:id/visibility | body: VisibilitySettings |

## 8. 异常处理

| 异常场景 | 处理方式 |
|----------|----------|
| 验证码发送失败 | Toast "发送失败，请稍后重试"，60s 后可重新获取 |
| 验证码错误 | 提示"验证码错误"，剩余尝试次数 |
| 邀请码过期 | 提示"邀请码已过期，请联系家人重新邀请" |
| 邀请码无效 | 提示"邀请码不正确，请检查" |
| 家庭圈已达10人上限 | 提示"家庭圈最多容纳 10 人" |
| 用户已在其他家庭圈 | 提示"您已加入其他家庭圈，切换到新家庭将退出原家庭" |
| 网络异常 | 关键步骤支持离线暂存，恢复网络后自动提交 |
