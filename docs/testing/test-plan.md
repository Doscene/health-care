#### 3.1.1 注册流程

| 用例ID | 测试场景 | 测试步骤 | 预期结果 | 优先级 |
|--------|----------|----------|----------|--------|
| AUTH-001 | 手机号注册成功 | 1. 输入手机号 2. 获取验证码 3. 输入验证码 4. 提交 | 注册成功，返回 JWT Token | P0 |
| AUTH-002 | 手机号格式错误 | 1. 输入无效手机号 2. 点击获取验证码 | 提示"手机号格式错误" | P0 |
| AUTH-003 | 验证码错误 | 1. 输入正确手机号 2. 输入错误验证码 | 提示"验证码错误"，显示剩余尝试次数 | P0 |
| AUTH-004 | 验证码过期 | 1. 获取验证码 2. 等待超时 3. 输入验证码 | 提示"验证码已过期，请重新获取" | P0 |
| AUTH-005 | 验证码发送频率限制 | 1. 获取验证码 2. 60秒内再次获取 | 提示"请60秒后重试" | P0 |
| AUTH-006 | 重复注册 | 1. 使用已注册手机号注册 | 自动转为登录流程 | P1 |
| AUTH-007 | 角色选择-患者 | 1. 注册时选择"我是患者" 2. 填写姓名+出生年份 3. 选择病种 | 用户角色设置为 patient | P0 |
| AUTH-008 | 角色选择-家人 | 1. 注册时选择"我是家人" 2. 填写姓名 | 用户角色设置为 child | P0 |
